# 🌲 Cypress, from Zero to the Cloud ☁️

Sample project fot the "Cypress, from Zero to the Cloud" course of Talking About Testing online school.

## Pre-requirements
It is required to have git. Node.js and npm installed to clone and run this project.

> I've used versions `2.47.9`, `v20.20.0` and `10.8.2` of git, Node.js and npm, respectively. I suggest you use the same or later LTS versions.

## Installation
Run`npm install` (or `npm i` for the short version) to install the dev dependencies.

## Tests
In this project, you can run tests on desktop or mobile viewport.

### Desktok
Run`npm test` (or `npm t` for the short version) to run test in headless mode on desktop viewport.

Or, run `npm run cy:open` to open the Cypress App on desktop viewport.

### Mobile

Run`npm test:mobile` to run test in headless mode on mobile viewport.

Or, run `npm run cy:open:mobile` to open the Cypress App on mobile viewport.
